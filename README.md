# Data Visualization with Excel — Car Sales Dataset

## Project Overview

Hands-on data visualization project using **Microsoft Excel**, working with a real-world car sales dataset from Kaggle. Built 4 different chart types to extract and communicate insights about sales performance, pricing, and customer retention across Japanese auto manufacturers.

This project was developed as part of the **IBM Data Analyst Professional Certificate** on Coursera, specifically the course [Data Visualization and Dashboards with Excel and Cognos](https://www.coursera.org/learn/data-visualization-dashboards-excel-cognos/).

---

## Business Questions Answered

- Which Toyota model has the highest Power Performance Factor?
- How does the average sale price compare to the 1-year resale value across Toyota models?
- Which Japanese manufacturer retains the most value over time?

---

## Charts Built

### 📌 Project 1 — Power Performance Factor by Toyota Model
| | |
|---|---|
| **Chart Type** | Column Chart (Clustered 2D) + Area Chart (2D) |
| **What was analyzed** | Power Performance Factor across 9 Toyota models |
| **How it was built** | Applied column filters to segment data by manufacturer; customized series fill (gold), data labels, and chart title via the Chart tab |
| **💡 Key Insight** | The Land Cruiser has the highest Power Performance Factor among all Toyota models, nearly double the Corolla's value |

---

### 📌 Project 2 — Average Sale Price vs. 1-Year Resale Value
| | |
|---|---|
| **Chart Type** | Clustered Bar Chart (2D) from a Pivot Table |
| **What was analyzed** | Average Sale Price vs. Average 1-Year Resale Value across Toyota models |
| **How it was built** | Built from a Pivot Table summarizing aggregated values per model; expanded fields to show model-level detail; added Inside End data labels for clear value readability |
| **💡 Key Insight** | The RAV4 shows the largest gap between sale price and resale value among Toyota models, while the Corolla retains a proportionally higher percentage of its value |

---

### 📌 Project 3 — Average Retention % by Japanese Manufacturer
| | |
|---|---|
| **Chart Type** | Line Chart with Markers (2D) |
| **What was analyzed** | Average retention % across 8 Japanese manufacturers (Acura, Honda, Infiniti, Lexus, Mitsubishi, Nissan, Subaru, Toyota) |
| **How it was built** | Used a Pivot Table as data source; filtered by manufacturer; customized title and data labels |
| **💡 Key Insight** | Lexus leads with 81% retention; Nissan is the lowest at 59%; overall average sits at 72% |
